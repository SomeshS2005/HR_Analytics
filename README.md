# HR Analytics Dashboard

## Project Overview

This project analyzes employee workforce data to provide insights into employee attrition, department distribution, job satisfaction, salary trends, and workforce demographics.

## Tools Used

* Power BI
* SQL
* MySQL

## Key Metrics

* Total Employees
* Attrition Count
* Attrition Rate
* Average Salary

## Dashboard Features

* Employees by Department
* Attrition by Department
* Gender Distribution
* Job Satisfaction Analysis
* Interactive Filters

## Key Insights

* Research & Development has the highest employee count.
* R&D also experiences the highest attrition.
* Most employees report high job satisfaction.
* Male employees represent the majority of the workforce.

## Skills Demonstrated

* Data Cleaning
* Data Analysis
* SQL Querying
* Data Visualization
* Business Intelligence
